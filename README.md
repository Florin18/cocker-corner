# 🐾 Cocker Spaniel Englez — Single Page Website

Website static modern, construit într-un singur fișier `index.html`, dedicat rasei **Cocker Spaniel Englez**.

Proiectul este realizat ca exercițiu de front-end și UI/UX, demonstrând structură clară, design coerent și JavaScript minimal.

---

## 📌 Descriere

Acest website oferă un ghid concis și bine structurat despre rasa Cocker Spaniel Englez, incluzând:

- Prezentare generală
- Temperament
- Nevoi zilnice
- Mișcare
- Stimulare mentală
- Îngrijire
- Socializare
- Test interactiv de potrivire
- Secțiune detaliată (accordion)
- Contact prin Instagram

Totul este implementat într-un singur fișier HTML, fără framework-uri sau build tools.

---

## 🧱 Tehnologii utilizate

- HTML5 semantic
- CSS modern (custom properties, grid, responsive layout)
- Vanilla JavaScript
- Google Fonts:
  - Space Grotesk (headings)
  - Inter (body)

Fără:
- React / Vue / Angular
- jQuery
- Librării UI
- CMS

---

## 🧠 Funcționalități principale

### 1. Hero Section
- Headline clar
- CTA-uri
- Panou informativ „Pe scurt”
- Layout responsive pe 2 coloane

### 2. Navigație Sticky
- Rămâne vizibilă la scroll
- Evidențiază automat secțiunea activă
- Scroll smooth
- Offset corect pentru a nu acoperi titlurile

### 3. Card Grid Responsive
Secțiuni organizate în carduri:
- Mișcare
- Stimulare mentală
- Îngrijire
- Socializare

### 4. Test Interactiv (Vanilla JS)
„Este rasa potrivită pentru tine?”

- Checkboxes
- Evaluare automată
- Bară de progres
- Interpretare dinamică
- Buton reset

### 5. Secțiune Detaliată (Accordion)
Implementată cu `<details>` și `<summary>`:
- Accesibilă nativ
- Fără librării externe
- Micro-animații discrete

### 6. Contact
Comunicare prin Instagram:
@pawsomedora

## ⚙️ Structură fișier

Proiectul conține un singur fișier:

index.html

Structură internă:

<head>
  - Meta tags
  - Fonts
  - <style>
</head>

<body>
  - Header (sticky nav)
  - Hero
  - Secțiuni principale
  - Test interactiv
  - Secțiune detaliată
  - Contact
  - Footer
  - <script>
</body>

Tot CSS-ul este în <style>, iar tot JavaScript-ul în <script>.

## 🚀 Cum se folosește

1. Deschide fișierul `index.html` în browser.
2. Nu este necesară instalare, server sau build.
